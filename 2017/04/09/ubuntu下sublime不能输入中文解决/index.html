本文适用于ubuntu16

让sublime支持gbk编码

安装convertToUTF8插件
首先，按住ctrl+shift+p，
然后，在弹出的搜索框中输入install package
最后，输入ConvertToUTF8，等待安装
安装完成后重启subline Text3即可。
